### taskcalendar

A full-stack task management application that integrates task tracking with calendar-based scheduling.

## Features

- Task creation, editing, deletion
- Drag-and-drop calendar scheduling
- Task dependencies & subtasks
- Recurring tasks
- Google Calendar integration
- Dark mode
- Offline support (partial)

## Tech Stack

**Frontend**

- React
- Tailwind CSS
- FullCalendar

**Backend**

- Node.js
- Express.js
- MongoDB
- Auth0
